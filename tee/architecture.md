## TEE Architecture (High-Level)

1. Input: Study objective and constraints
2. Orchestration Layer:
   - Routes tasks to specialized agents
3. Agent Examples:
   - Study Design Agent
   - Analysis Plan Agent
   - Code Generation Agent
   - Evidence Summary Agent
4. Optional RAG Layer (internal only):
   - Grounds outputs in trusted evidence
5. Outputs:
   - Draft study documentation
   - Analytic code templates
   - Structured summaries

All outputs are assistive and require scientific review.
